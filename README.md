<h1>conf_szh</h1>
<p>This tool used to configuration linux server.<br/>Enjoy ^_^</p>
<h1>How to install :</h1>

```
git clone https://github.com/zangestu2Eszh/conf_szh
cd conf_szh
chmod +x *
```
<h1>Running:</h1>
<b>If You Have Linux 32bit :</b><br/>

```
sudo ./conf_szh32
```
<b>If You Have Linux 64bit :</b><br/>

```
sudo ./conf_szh64
```
